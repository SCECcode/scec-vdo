CFM 5.0 release (limited)
-------------------------

This archive contains version 5.0 of the SCEC Community Fault Model (CFM) for southern California. Fault representation are grouped into a primary set considered the currently most appropriate model for many purposes by the CFM architects, and into an alternative set for faults where multiple representations exist for historic and interpretational reasons. The tsurf fault data files are supported by a comprehensive database in form of spreadsheet which defines a hierarchy of faults with many levels.
