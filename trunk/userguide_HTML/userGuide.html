Menu Options

  • While working on a project, users can save the current state of SCEC-VDO and resume at a later time by selecting File -> Save state.

  • To resume a project from an earlier time, select File > Open. Locate and select the name of your project.

  • By selecting Display, users have access to the following plugins:

    • Earthquake> Earthquake Catalog
    • Faults > Community Fault Model
    • GIS Hazus Events > GIS Hazus Events
    • Earthquake> Legend > Legend
    • Earthquake> Earthquake Simulators > Earthquake Simulators
    • Earthquake> ShakeMap > ShakeMap
    • Earthquake> Surface -> Surface.

  • Creating animations in SCEC-VDO is controlled by the Timeline plugin which allows users to display data from plugins onto the map. To access this plugin, select Render > Timeline.

3-D Viewer

  • The 3-D Viewer enables the user to view data both above and below the Earth’s surface at any angle.

  • Changes made through plugins are automatically updated onto the map.

Plugin Panel

  • Plugins enable the user to display and manipulate information onto their project.

  • All active plugins can be found on the right panel of SCEC-VDO.

  • Active plugins are identified with a blue highlight at the top of the panel.

  • All changes made within a plugin will automatically update the project and display on the 3-D Viewer.


    • Note: Each plugin is described in detail in the plugin section.

Focal Point

    The focal point is the center of the user’s perspective. When navigating SCEC-VDO, users are able to go beneath the earth’s surface and view earthquakes in depth.

    Add a focal point:Display>Training>Focal Point.

    In the case where the user is lost or out of perspective, press the ESC button to return to the default view.


    Controls

  • ESC: Reset Focal Point
  • Scroll Wheel Click + Drag:: Translate Focal Point
  • Left Click + Drag:: Rotate About Focal Point
  • Right Click + Drag Or Scroll Wheel:: Zoom In/Out
  • CTRL + Left Click Drag:: Roll Map


    Note: Losing your focal point occurs very frequently.

Surface

    The 3-D structure illustrates major faults in Southern California. Vertical faults such as the San Andreas (yellow band from top left to bottom right) are shown as a thin strip.***

Shake Map

    What is a ShakeMap?


    A ShakeMap displays the motion and intensity of an earthquake onto a map focused on the epicenter.


    Display preloaded ShakeMaps within California onto current project or import new maps from the USGS website.


    How to Open ShakeMap Plugin:

    Select Display> ShakeMap> ShakeMap Plugin

Load ShakeMaps from USGS

  1. Loading USGS Website

    2. Within the ShakeMap Plugin, select the Download USGS Map option, then Open the USGS Website.

  2. Accessing ShakeMap Database

    3. Select Search ShakeMap Archives.


    **Requirement The current iteration of SCEC-VDO only supports ShakeMaps in California.

  3. ShakeMap Search

    4. Customize your ShakeMap Search. This can include a specific magnitude, date, and region.

  4. ShakeMap Information

    5. Select ShakeMap from the results.

  5. ShakeMap Event ID

    6. Locate and copy the unique event ID located above the ShakeMap image.

  6. Event ID in ShakeMap Plugin

    7. Paste the event ID in the ShakeMap Plugin. Ensure the appropriate region is selected and then Download USGS Shake Map.

  7. Load Imported ShakeMap

    8. Imported ShakeMaps can be found in the Load Map tab within the ShakeMap Plugin.

    Political Boundaries are the outline of states and continents to distinguish between areas.

Graticule

    What is Graticule?

    The Graticule is a network of lines representing meridians and parallels on the map.

    Modify grid and background colors, dimensions, and text on map.


    How to Open Graticule:

    Select Graticule within Plugin Panel upon launching SCEC-VDO.



    ***“A graticule is a network of evenly spaced horizontal and vertical lines used to identify locations on a map.” Many programs, such as ArcGIS, utilize graticules to measure locations using projected coordinates.

Earthquake Simulators

    What is an Earthquake Simulator?

    Earthquake simulators replicate seismic activity during an earthquake in order for scientists to analyze key characteristics such as aftershocks, foreshocks, amplification, and elastic rebound.

    Load, modify, and animate earthquake fault lines on SCEC-VDO showcasing rupture sequences, magnitude, and length.


    Displaying an Earthquake Catalog :
    Select Display > OpenSHA > Earthquake Simulators

    ***Earthquake Simulators enable scientists to visualize an earthquake propagating through a fault. “The 2008 Great Southern California ShakeOut was based on a comprehensive scenario involving a magnitude 7.8 earthquake along the southernmost San Andreas fault.“


    1. Select File

      2. The Earthquake Simulators Plugin comes preloaded with several packages to choose from. Click on the desired file and ensure the visibility is toggled ON by checking the box.

      To import a new file, selectd the arrow pointing downward to access the drop-down list and scroll to Select File. Once the file has been selected, ensure the visibility is toggled ON by checking the box.

      For this example, the catalog used is the UCERF3 model.


    2. Color

      3. Adjust the appearance of the fault model by modifying the color scheme. Choose between a wide range of colors by selecting the color wheel option.


      Adjust the Color Faults By:

      • Slip Rate (mm/yr)
      • Simulator Participation Rates
      • Depth (km)
      • Patch Scalar File Loader
      • Simulator Output Animation: Slip (m) Per Event
      • Custom


    3. Display Settings

      4. Adjust the look of the fault model by modifying fault attributes.

      Geometry Type:
      • Lines
      • Points
      • Polygons


      Surface Type:
      • All Lines
      • Outline Only
      • Trace Only


    4. Animation

      5. Modify earthquake animations based on:

      • Duration: Length of the video.
      • Animation Type (Time Based or Evenly Spaced): Time Based animations occur at the time of the earthquake, where as evenly spaced animation spread the earthquakes evenly allowing to visually differentiate between each earthquake.
      • Min Mag: Animate earthquakes showing only with a specific magnitude and above.
      • Only Events Involving: Animate only a certain area of a fault.
      • Filter By Fault: Select only a specific fault to animate.
      • Start Event ID: Begin animation starting with an earthquake’s unique ID.

Earthquake Catalog

    What is an Earthquake Catalog?

    Earthquake Catalogs contain earthquake source parameters (e.q. hypocenters, magnitudes, phase picks and amplitudes) and other products (e.g. moment tensor solutions, macroseismic information, tectonic summaries, maps) produced by contributing seismic networks. 1


    Display multiple earthquake catalogs onto SCEC-VDO map.


    How to Open Earthquake Catalog:

    Select Display > Earthquake > Earthquake Catalog Plugin



    1. Loading Earthquake Catalogs

      2. Check the boxes of Earthquake Catalogs to display onto the map.

    2. Adjusting Appearance

      3. Select Display within the Earthquake Catalog Plugin. Modify the shape, color, or transparency of each Earthquake Catalog.


    Note: Loading multiple catalogs at a time may cause a slow down in performance.



    **An earthquake catalog is a list of earthquakes with information about them, usually the time and location that they occurred and their seismic moment or magnitude.

Legend Catalog

    What is a Legend?

    A legend is an explanation of tables, diagrams, charts, or other forms of information.


    Display an on-screen legend for SCECVDO animations to inform viewers on slip rates and magnitudes.


    How to Open Legend:

    Select Display > Legend > Legend Plugin



    1. Create Legend

      2. Select Create Legend

    2. Select Orientation

      3. The orientation allows the user to display either the color then text, text then color, or a gradient on the legend.


    3. Naming the Legend

      4. Enter the name of the legend within the text field. After naming the legend, select Add to List


    4. Saving the Legend

      5. Once the legend is complete, select Create Legend. This will save the legend as a JPEG file.


    5. Displaying the Legend

      6. After saving the legend, select Add Image and open the legend file.


      Some plugins, such as Earthquake Simulators, may a legend option built in to describe a certain function.





    Legends describe what a particular line, symbol, dot, or color represents on a map. In SCEC-VDO, the magnitude of slip can easily be differentiated throughout the San Andreas fault by displaying a legend.

Scripting Controls

    Scripting is the process of utilizing the timeline plugin in SCEC-VDO to create three-dimensional animations.

    What are key frames?

    • Key frames enable the user to transition between positions or toggle data on/off on the map at a certain time and for a specific duration.
    • Key frames are separated by plugins – Camera, Political Boundaries, Graticule, and so on.




    Insert Key Frames –Insert key frames in the Timeline Plugin either by double clicking at a particular time, or from the key frame options.


    Delete Key Frames –Delete a key frame either by right clickingthe mouse or using the scroll wheel clickon a particular key frame.


    Camera Position Key Frame –Change the position of the map from one position key frame to another.


    Camera Pause Key Frame –Pause the next position key frame in place.


    Show/Hide Entire Plugin –Show or hide key frame data for a plugin during an animation.


    Freeze Plugin –Freeze plugin key frame data which can increase CPU performance.


    Normal Key Frame – Normal Key Frames for Stateful plugins.


    Visibility On Key Frame – The visibility on key frame allows the user to make plugins visible on the map during an animation at a specific point in time.


    Visibility Off Key Frame – The visibility off key frame allows the user to make plugins invisible on the map during an animation at a specific point in time.


    Render Settings

    • Change the duration of the animation in seconds.


    • Adjust the camera spline to alter the transition behavior between position key frames.
      • Cardinal
      • Kochanek
      • S Curve


    • Modify rendering output file.
      • MPEG-4, H.246, Lossy (Default)
      • MPEG-4, PNG Lossless (Higher Quality, Less portable)
      • MPEG-4, JPEG
      • PNG Sequence
      • JPEG Sequence


    • Adjust output resolution


    The Timeline plugin enables the user to insert key frames in a predetermined amount of time to animate various datasets, angles, and perspectives. This process is known as scripting.

Creating Animations

    Transition Key Frames –For this example, the camera key frames will move the map from the default position to a zoomed in, tilted position.


    1. Place a normal key frame in the camera section of the Timeline, as shown in picture A.


    2. Tilt, zoom, and drag the map to another position, such as picture B, and add another key frame. Keep in mind the amount of time in between key frames. The closer key frames are positioned to another, the faster the animation will be.

      3. Toggle Data On/Off –Toggling data on/off allows the user to have specific data appear, disappear, freeze or play at a certain point in the animation. In this example, the animation will be begin with no interstates in California, and after 5 seconds, they will become visible.


    3. Access the Drawing Tools plugin.


    4. Turn ON California Interstates


    5. Access the Timeline plugin


    6. Insert a Visibility Off Key Frame corresponding to the Drawing Tools plugin in the Timeline. The visibility off key frame will take the data set in that specific plugin and make it non-visible.


    7. Place a Visibility On key frame at the 5 second mark. If you place another key frame at the same time location, it might be under the active key frame, simply slide over the current key frame in order to reveal the new key frame.
